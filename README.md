## Pack and Go

### Run

1. ```bash
   git clone https://github.com/farnazhaghshenas/pag.git
    ```

2. ```bash
   make make app-config
   ```

3. ```bash
   make install_dependencies
    ```

4. run migrations
   
5. ```bash
   go run main.go seed
    ```
   
6. ```bash
   go run main.go serve
    ```
